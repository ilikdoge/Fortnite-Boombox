# fortnite boombox

old project

## building and running

<code>
git clone https://github.com/ilikdoge/Fortnite-Boombox.git
cd Fortnite-Boombox
git submodule update --init --recursive
npm install
</code>

### running

Set your bot's token in <code>bot_files/config.json</code>

<code>node index.js</code>
